# Changelog

All notable changes to ToposAI should be recorded here.

The format is inspired by Keep a Changelog, and this project uses semantic
versioning once public releases begin.

## [Unreleased]

- Release guide with clean-workspace and wheel-install checks.
- Benchmark reporting template for reproducible measurements.
- Project status documentation describing tested and experimental areas.
